HOW TO USE MY API

Request URL http method
Return a list of ALL movies to the user /movies GET
Return data about a genre movies/genre/:name GET
Return data about a director by name movies/director/:name GET
Allow new users to register /users POST
Allow users to update their user info /users/:id PUT
Allow users to add a movie to their list of favorites /favorites/:username/:movieId DELETE
Allow users to remove a movie from their list of favorites /favorites/:username/:movieId DELETE
Allow existing users to deregister /users/:username DELETE
